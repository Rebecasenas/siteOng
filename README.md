# 🥭 FrutaViva — Projeto Acadêmico com Cliente Real

Este site foi desenvolvido por uma equipe de **4 integrantes** durante um projeto acadêmico com duração de **3 meses**.

O desafio proposto era claro: **encontrar um cliente real e desenvolver uma solução digital alinhada ao seu negócio.**

O cliente escolhido foi o proprietário de um hortifruti local, pai de uma das integrantes da equipe. A partir disso, realizamos o levantamento de necessidades, entendemos o público-alvo e desenvolvemos um site institucional simples, objetivo e funcional.

## 🎯 Objetivo do Projeto

Criar uma presença digital para o hortifruti, facilitando o contato com clientes e fortalecendo a divulgação do negócio.

---

## 💡 Solução Desenvolvida

O site apresenta informações sobre o hortifruti e direciona o usuário de forma estratégica para o contato direto com o proprietário, permitindo que clientes realizem pedidos ou tirem dúvidas com facilidade.
